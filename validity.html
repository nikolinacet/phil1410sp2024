Are the following arguments valid?

Argument 1:

1. All men are mortal.
2. Socrates is a man.
∴ Socrates is mortal.

Valid

Argument 2:

1. All men are mortal.
2. Socrates is mortal.
∴ Socrates is a man.

Invalid

Argument 3:

1. If geckos are mammals, then geckos are vertebrates.
2. Geckos are not mammals.
∴ Geckos are not vertebrates.

Invalid

Argument 4:

1. It’s going to be rainy and cold tomorrow.
∴ It’s going to be rainy tomorrow.

Valid

Argument 5:

1. Exercise is good for you.
∴ Exercise is not not good for you.

Valid

Argument 6:

1. It is not the case that there is no God.
∴ So, surely, God does exist.
