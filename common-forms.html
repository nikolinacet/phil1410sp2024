As arguments get longer, their truth tables have more rows. Using truth tables to determine their validity can become quite time-consuming. For example, the truth table of the following argument has 16 rows, and can take quite a bit of time to construct.
If interest rates are raised, the stock market will be hurt. If the stock market is hurt, the economy will slow down. But if interest rates are not raised, inflation will get worse. If inflation gets worse, the economy will slow down. So the economy will slow down.
Since using truth tables to determine validity can be tedious, there is an incentive to find a more efficient way.

Arguments such as the one above are built by combining small and basic valid argument forms. This means that if we can recognize the small forms and see how they are put together to form longer arguments, then we can determine validity without constructing truth tables.

Use the list of common argument forms below to identify the form of the arguments which follow. For each argument, determine if it is valid.

Example:

1. If it rains, then I won't go out.
2. I went out.
∴ Therefore, it didn't rain.

R = It rains
O = I go out/went out

1. P ⊃ ~O
2. O
∴ ~P
