Deductive Argument Reconstruction

Home Deductive Arguments
Here is one strategy for reconstructing and evaluating deductive arguments.

Step 1 Identify the premise(s) and conclusion(s) in standard form. Start with the conclusion.
Step 2. Omit any extraneous details. E.g. repetition, tangents, guarding terms, assuring terms.
Step 3. Clarify where necessary. E.g. Replace pronouns with proper nouns, split up sentences.
Step 4. Represent the parts of the argument in a chain or tree.
Step 5. Assess each argument and subargument for validity. Add suppressed premises if necessary.
Step 6. Assess the truth of the premises.

The steps above are a procedural guide. They do not need to be followed in exactly the same order every time. For example, if you add a suppressed premise in step 5, you'll have to go back to step 4 and create a new argument chain or tree to include the suppressed premise, so in some cases, it may make more sense to attempt step 5 before step 4.
